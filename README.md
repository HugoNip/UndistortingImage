# Undistorting Image
## Introduction
This project shows how to correct image for lens distortion. 
## Requirements
### OpenCV
#### Required Packages
OpenCV  
OpenCV Contrib

## Compile this Project
```
mkdir build
cd build
cmake ..
make 
```

## Run
```
./undistortImage
```
